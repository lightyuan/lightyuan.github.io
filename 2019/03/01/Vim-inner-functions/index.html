Vim inner functions

发表于 2019-03-01

又换了个系统，突然发现Vim又要重新配置了，惨
在此记录下常用的一些内置函数

常用内置函数

substitute 替换子字符串函数

substitute({expr}, {pat}, {sub}, {flags})
和s命令一样,只不过这里要处理的是字符串{expr}, 其中{flags}可以是g(替换全部), 在 :e_flags 查阅详细用途
返回字符串

tr 替换字符函数

tr({src}, {fromstr}, {tostr})
具体用法如下

返回字符串

getline 获取指定行内容

getline({lnum} [, {end}])
有两种调用方法

  1. 获取某一行的内容 getline({lnum}), 返回string
  2. 获取连续多行的内容 getline({lnum}, [, {end}]), 返回List

expand 获取文件名

expand({expr} [, {nosuf} [, {list}]])
{expr}可为 %:
% current file name
可以搭配修饰符:
